Módulo Odoo – Gestión de Propiedades Patrimoniales

Municipalidad de Pérez Zeledón

Descripción

---

Este repositorio contiene el módulo personalizado property_patrimonial, desarrollado para la gestión de:

Propiedades patrimoniales

Zonas verdes

Expedientes asociados

Registro y mantenimiento de información municipal

El módulo está diseñado para integrarse en una instalación estándar de Odoo.

Requisitos

---

Odoo (web)

Python 3.11.7 (no usar versión superior a 14)

PostgreSQL

Instalación

---

1. Clonar el repositorio: git clone https://github.com/MarisolValverde07/muni_propiedades_odoo.git
2. Copiar la carpeta property_patrimonial dentro del directorio de módulos personalizados (custom_addons o equivalente).
3. Agregar la ruta en el archivo odoo.conf:addons_path = addons,custom_addons
4. Reiniciar el servidor Odoo.
5. Activar modo desarrollador y actualizar lista de aplicaciones.
6. Instalar el módulo Property Patrimonial.
